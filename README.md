# 双人五子棋

一个适合手机浏览器和微信打开的双人网页小游戏，包含五子棋和双人斗地主。

## 免费发布到 GitHub Pages

1. 登录 https://github.com
2. 新建公开仓库，比如 `gobang`
3. 上传本目录里的全部文件
4. 进入仓库 `Settings` -> `Pages`
5. `Build and deployment` 选择 `Deploy from a branch`
6. Branch 选择 `main` 和 `/root`，保存
7. 等待 1 分钟左右，访问 `https://你的用户名.github.io/gobang/`

## 玩法

- 先选择五子棋或双人斗地主。
- 创建房间后，把邀请链接通过微信发给对方。
- 右上角小圆点是隐藏管理员入口，输入 `1115` 可开启突破限制。
- 五子棋管理员可以设置本回合下几颗。
- 斗地主管理员可以无视牌型和大小，想出什么牌就出什么牌。
